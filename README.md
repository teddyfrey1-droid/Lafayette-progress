# Lafayette Progress (clean structure)

## Structure

This repo is intentionally kept simple to avoid deployment bugs:

- `index.html` **(single entry point)**
- `css/` stylesheets
- `js/` scripts (no build step)
- `pages/` extra pages (Contacts, Sites utiles, etc.)

### Rule: only one index

Do **not** add `Index.html` or `index1.html`... keep only `index.html`.
